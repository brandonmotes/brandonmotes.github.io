This lab is working to develop a table top MRI machine. They needed a device that can map the magnetic field of the array of permanant magnets. I designed a PCB containing holding an array of hall effect sensors on a I2C bus connected to a Teensy that can be connected to a PC via a USB cable. I then modified a 3D printer by extending the y axis and replacing the head with an arm that holds the sensor board.

I also wrote software which takes the dimensions of the magneta and the desired density of measurements. The software will generate an efficient path to utilize all of the sensors to reduce run time. The data is then exported to a spread shett and a graph can be generated to visualize the field.
